      ❓ User Guide
      1

      Select Prediction Indicator

      On the "Single Sample" page, select the clinical indicator of interest (e.g., BMI, body fat percentage) from the model cards on the left. Each indicator has quartile and tertile grouping models.

      2

      Input Lipidomics Data

      Fill in the baseline lipidomics detection values in the right panel. You may use the "Use Example" button to auto-fill with mean values from the training cohort, or manually enter your data. Only the lipids used by the selected model need to be filled; others may be left blank.

      3

      Get Prediction Result

      Click "Predict" to receive either "High Response" (significant exercise effect) or "Low Response" (weaker exercise effect) classification, along with probability values and confidence scores. Model formula and characteristics are displayed above the input form.

      4

      Batch Prediction

      To predict multiple samples at once, upload a CSV file (each column corresponds to a lipid molecule, each row to a different sample). The system returns all predictions and supports CSV download.

      Δ indicators
      Δ = outroll - enroll, so the current website predicts change-response targets rather than absolute one-time measurements.
      Q / T grouping
      Q = Q1 vs Q4 and T = T1 vs T3. The middle quantiles are excluded from the current extreme-group binary models.
      Response direction
      For negative-better indicators such as ΔBMI / ΔPBF / ΔWHR, high response means decreasing more. For ΔPSM, high response means increasing more.
