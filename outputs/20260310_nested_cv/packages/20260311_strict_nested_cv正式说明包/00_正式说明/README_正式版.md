## 本包的正式定位

- 正式主结果采用 **strict nested CV** 口径。
- 现阶段可以支持的结论是：**存在一定预测信号，但泛化增益有限；结果可信度优先于分数高低。**
- 先前较高分结果如需提及，只能作为 **开发阶段探索性结果**，不作为本包的正式主结果。
