# 训练 AUC 与泛化差距补充图说明

本目录图件用于补充展示“训练 AUC 较高”这一现象，但正式汇报时不建议单独强调训练 AUC，而应与外层测试 AUC 一并呈现。

## 推荐使用顺序

1. `20260311_训练AUC与泛化差距_组合图.png`
   - 一页内同时展示训练 AUC、外层测试 AUC、泛化差距和模型复杂度。
2. `20260311_训练AUC_vs_外层测试AUC_条形图.png`
   - 最适合直观看到“训练 AUC 普遍更高”。
3. `20260311_泛化差距_dumbbell图.png`
   - 最适合说明不同模型的 train-test gap。
4. `20260311_AUC差值_vs_特征规模_气泡图.png`
   - 最适合说明“变量越多，不一定泛化越好，反而差值更大”。

## 正式表达建议

1. 可以强调：训练 AUC 普遍高于外层测试 AUC，提示当前模型已学到一定模式。
2. 更稳妥的同时表述是：训练 AUC 与外层测试 AUC 之间存在明显差距，说明现阶段仍需优先关注过拟合与泛化稳定性。
3. 若用于正式说明，建议优先使用“组合图”或“dumbbell 图”，避免只放训练 AUC 单边图。
